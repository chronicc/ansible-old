# Ansible Cheatsheet

## Variables

### Ansible Version

Ansible provides a global dict called `ansible_version`.

```json
"ansible_version": {
    "full": "2.7.4",
    "major": 2,
    "minor": 7,
    "revision": 4,
    "string": "2.7.4"
}
```
